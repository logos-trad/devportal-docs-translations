## How is it used?

#### Websites and App

#### Specifications

The ‚ÄúSign with IO‚Äù button is comprised of three main elements that cannot be changed: logo, font and color.

#### Logo

The button must always contain the IO logo, without changes to the proportions or distortions of the dimensions.

#### Font

The text ‚ÄúSign with IO‚Äù must always be present next to the IO logo. The font used for it must be Titilium Web Bold with body 14 px, letter spacing 0.3 px

#### Color

The background color of the ‚ÄúSign with IO‚Äù button must be Main Blue #0073E6, the applied elements, such as the logo and text, must be displayed in white.

#### Dimensions

It is possible to enlarge the button based on visual requirements. The following dimensions cannot be changed: the height is 40px, upper and lower padding is 11 px and finally the rounding radius of the corners of the button is 4 px.
